# 🚀 Cognifyz Machine Learning Internship Projects

This repository contains completed Machine Learning internship tasks developed as part of the Cognifyz Technologies Internship Program.

The projects focus on:
- Machine Learning
- Data Analysis
- Recommendation Systems
- Data Visualization
- Geographical Analysis

---

# 📂 Projects Included

## 1️⃣ Restaurant Rating Prediction

### 📌 Objective
Build a machine learning model to predict restaurant ratings based on restaurant features.

### ✅ Features
- Data preprocessing
- Missing value handling
- Feature selection
- Linear Regression model
- Model evaluation using:
  - Mean Squared Error (MSE)
  - R² Score
- Graph visualization

### 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

---

## 2️⃣ Restaurant Recommendation System

### 📌 Objective
Create a recommendation system that suggests restaurants based on user preferences.

### ✅ Features
- Cuisine-based filtering
- Price range filtering
- Personalized recommendations
- Rating-based sorting
- Recommendation visualization

### 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

---

## 3️⃣ Location-Based Restaurant Analysis

### 📌 Objective
Perform geographical analysis of restaurants using location data.

### ✅ Features

- City-wise analysis

### 🛠️ Technologies Used

- Folium

# 🚀 How to Run the Projects

## Step 1
Clone the repository:

```bash
git clone https://github.com/Ashwitha1362/Cognifyz_ML_Internship.git
```

## Step 2
Open the project folder in VS Code.

## Step 3
Install required libraries:

```bash
pip install pandas matplotlib scikit-learn folium
```

## Step 4
Run any project:

```bash
python main.py
```
